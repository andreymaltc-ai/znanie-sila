# Хочу всё знать — полный контекст проекта

## Что это
Telegram Mini App — геймифицированные квесты по Калининграду и Сочи (механика Pokémon GO + реальная природа/история).
Пользователь гуляет, фотографирует растения/места, получает факты, выполняет квесты.
Аудитория: дети + семьи + эксперты. Монетизация: B2C подписка + B2B школы + гранты.

**Владелец проекта:** Андрей (г. Калининград, связан с брендом Понарт).

## Деплой
```bash
vercel --prod
```
VERCEL_TOKEN уже в переменной окружения — авторизация не нужна.
После деплоя автоматически обновляется https://znanie-sila.vercel.app

## Стек
- Весь фронтенд: `public/index.html` — ОДИН файл, Leaflet.js карта, vanilla JS
- `api/identify.js` — Claude Vision API для определения растений по фото
- `api/webhook.js` — Telegram webhook
- `lib/claude.js` — Anthropic SDK обёртка
- `.env` — CLAUDE_API_KEY и другие ключи (не трогать, не коммитить)

## Правила разработки
- **Весь фронтенд в одном файле** public/index.html — не разносить
- После любого изменения: `vercel --prod`
- Кириллица: UTF-8, проверять после деплоя
- GPS координаты: брать из 2ГИС / Яндекс.Карты, не выдумывать
- Отвечать по-русски
